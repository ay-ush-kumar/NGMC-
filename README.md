# NGMC-
This 📌 NGMC Python Audit Project
🧾 Overview
This project presents a practical audit of Python as a Free and Open Source Software (FOSS) tool within a Linux environment. It combines theoretical insights with hands-on Bash scripting to analyze system behavior, security, and software structure.
🎯 Objectives
Understand Python’s role in the FOSS ecosystem
Explore Linux file system structure
Perform system-level auditing using shell scripts
Analyze security practices and permissions
Demonstrate real-world integration of Python
🛠️ Technologies Used
Linux (Ubuntu/Kali/Any Distro)
Bash Shell Scripting
Python 3
Git & GitHub

⚙️ Scripts Description
🔹 Script 1 – System Handshake
Displays system details like hostname, kernel version, and date.
🔹 Script 2 – Python Check
Verifies Python installation and shows version details.
🔹 Script 3 – Directory Audit
Analyzes system directories, disk usage, permissions, and ownership.
🔹 Script 4 – Log Scanner
Searches log files for error patterns and counts occurrences.
🔹 Script 5 – Philosophy Generator
Generates a personalized open-source philosophy statement.
